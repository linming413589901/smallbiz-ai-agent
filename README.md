# SmallBiz AI Agent — 小商家智能客服

> 以前要 30 分钟，现在 30 秒。让每个小店都拥有一个 24 小时在线的 AI 客服。

**[在线演示](https://smallbiz-ai-agent.vercel.app)** | **[技术博客](./blog-post.md)**

---

## 解决什么问题

小商家每天花大量时间重复回答顾客问题：有没有货？发货要几天？能退吗？

雇一个客服：每月 3000-5000 元，还不一定 24 小时在线。
用自动回复：死板，顾客问复杂问题就懵了。

**AI 智能客服 Agent 是第三种选择**：像真人一样聊天，能查库存、查订单、回答常见问题，7x24 小时在线，成本不到一杯咖啡。

## 谁在用

| 场景 | 痛点 | Agent 怎么帮 |
|------|------|-------------|
| 淘宝/拼多多店铺 | 每天回复上百条"有货吗""发什么快递" | 自动查库存、自动回复政策问题 |
| 线下餐饮店 | 顾客问营业时间、菜单、能不能预约 | 24小时自动回答，不漏一条消息 |
| 社区团购 | 团长半夜还要回消息 | Agent 值夜班，团长安心睡觉 |
| 本地服务（家政/维修） | 顾客问价格、能不能上门 | 自动报价、自动登记需求 |

## 核心能力

- **智能对话** — 理解自然语言，不是关键词匹配
- **商品查询** — 按名称、类别查库存和价格
- **订单跟踪** — 顾客报订单号，自动查状态和物流
- **FAQ 自动回复** — 营业时间、退换货、发货、包邮等
- **情绪识别** — 顾客不满时先安抚再解决问题
- **转人工兜底** — 解决不了的问题自动通知店主

## 技术架构

```
顾客消息 → Next.js API → LangChain Agent → MiMo AI 模型
                                           ↓
                              模型决策调用哪个工具
                                           ↓
                    ┌──────────┬───────────┴──────────┐
                    ↓          ↓                      ↓
               FAQ查询     库存查询               订单查询
                    ↓          ↓                      ↓
                    └──────────┴───────────┬──────────┘
                                           ↓
                              模型组织自然语言回复 → 顾客
```

**技术栈**：Next.js 14 + LangChain + MiMo API + Vercel

## 快速开始

```bash
git clone https://github.com/linming413589901/smallbiz-ai-agent.git
cd smallbiz-ai-agent
npm install
cp .env.example .env.local
# 编辑 .env.local，填入 API Key
npm run dev
```

打开 http://localhost:3001 即可体验。

## 商业价值

对于一个小商家来说：

- **省时间**：每天少花 2-3 小时回复重复问题
- **省 money**：比雇客服便宜 95%
- **不丢单**：半夜有人咨询也能秒回，不让顾客跑掉
- **可定制**：接入你自己的商品数据、FAQ、话术
